Projects

About the project:

The project began with the development of a profile website using HTML, CSS, and JavaScript. Following that, I created an STD (Software Test Description) document outlining test cases for the demo e-commerce site, Swag Labs. Swag Labs is a mock e-commerce platform where users can browse a product catalog and simulate purchases from various user profiles.
After documenting the test cases, I automated each one using Selenium with Java and TestNG. The tests were executed via Maven, and I analyzed the outcomes using Allure Reports. Any defects identified during testing were thoroughly documented in Jira, complete with screenshots and all necessary details to facilitate issue resolution.
To enhance test automation efficiency, I utilized Jenkins for scheduling tests at various intervals to ensure continuous functionality across time frames. Additionally, I ran the tests across multiple environments using Docker containers to validate platform consistency. The project concluded with the completion of an STR (Software Test Report) document summarizing the testing efforts.
